#### mac下chrome常用快捷键
开发者工具: cmd + option + J  
打开新的窗口: cmd + N  
打开新的标签页: cmd + T  
左右切换标签页: cmd + option + 左右  
打开书签管理器: cmd + option + B  
打开历史记录: cmd + y  
打开下载内容: cmd + shift + J  
打开设置: cmd + ,  
打开清除浏览器数据: cmd + shift + delete  
全屏: cmd + shift + F  

定位到地址栏: cmd + L  
地址栏中上或下一个关键字: option + 左右  
保存为书签: cmd + D
