### 查看远程分支信息
```
git remote -v
```

查看分支
```
git branch
```

查看远程分支
```
git branch -a
```
删除本地分支
```
git branch -d 本地分支名(在主分支中)
``

查看远程分支与本地分支的映射关系
```
git branch -vv
```
新建分支并自动切换
```
git checkout -b 本地分支名
```

情景：branchA中有文件B，branchB中有文件B，想用branchA中文件替换branchB中的文件B，切换到branchB，执行命令
git checkout branchA -- filename
即 git checkout branchA -- B
