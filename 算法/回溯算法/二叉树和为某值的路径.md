### 二叉树和为某值的路径
输入一颗二叉树的跟节点和一个整数，打印出二叉树中结点值的和为输入整数的所有路径  

回溯法：
是一种选优搜索法，又称为试探法，按选优条件向前搜索，以达到目标。但当探索到某一步时，发现原先选择并不优或达不到目标，就退回一步重新选择，这种走不通就退回再走的技术为回溯法，而满足回溯条件的某个状态的点称为“回溯点”  

- 设定一个结果数组来存储所有符合条件的路径
- 设定一个栈stack来存储当前路径种的节点
- 设定一个和sum来标识当前路径之和
    * 从根节点开始深度优先便利，每经过一个节点，将节点入栈
    * 到达叶子节点，且当前路径之和等于目标值，将结果加入数组
    * 遍历到二叉树的某个节点时可能有2个可能的选项，选择左子树或者右子树
    * 若存在左子树，继续向左子树遍历
    * 若存在右子树，继续向右子树遍历
    * 若上述条件均不满足，或已经遍历过，将当前节点出栈，向上回溯
```
function FindPath(root, target){
    const result = []
    if(root){
        FindPathCore(root, target, [], 0, result)
    }
    return result
}
function FindPathCore(node, target, stack, sum, result){
    stack.push(node.val)
    sum += node.val
    if(!node.left && !node.right && sum === target){
        result.push(stack.slice(0))
        return
    }
    if(node.left){
        FindPathCore(node.left, target, stack, sum, result)
    }
    if(node.right){
        FindPathCore(node.right, target, stack, sum, result)
    }
    stack.pop()
}
```
